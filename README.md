# Flutter Application

## Setup & Run

### Prerequisites
- Flutter SDK - version 3.44.4
- Dart SDK - version 3.12.2
- Android Studio or VS Code 
- Android Emulator or Physical Device

### Installation

```bash
flutter pub get
```

### Run the App

```bash
flutter run
```
