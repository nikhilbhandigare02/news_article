# Flutter Application

## Setup & Run

### Prerequisites
- Flutter SDK
- Dart SDK
- Android Studio or VS Code
- Android Emulator or Physical Device

### Installation

```bash
flutter pub get
```

### Run the App

```bash
flutter run
```

## Architecture
- **Architecture:** MVVM
- **State Management:** BLoC

## Dependencies
All required dependencies are listed in `pubspec.yaml`. Install them using:

```bash
flutter pub get
```
